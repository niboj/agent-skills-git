# Design

## Decisions

- mutualiser les regles Git transverses dans `docs/reference/` ;
- garder chaque `SKILL.md` court et directif ;
- stocker les exemples detaillees dans un emplacement source unique par identifiant ;
- utiliser `docs/governance/example-traceability-matrix.md` comme registre central ;
- limiter les scripts a des aides non interactives apportant une vraie reduction de risque ou de travail repetitif.
