# Proposition

## Titre

Ajouter une bibliotheque complete de skills Git pour Codex.

## Motivation

Le depot ne contenait pas de collection specialisee de skills Git couvrant l ensemble du cycle de travail agent : inspection, branchement, synchronisation, commit, revue, protection, recuperation, livraison, tag et hotfix.

## Portee

- 12 skills Git ciblees ;
- references partagees et gouvernance ;
- gabarit reutilisable ;
- scripts utilitaires limites aux cas de forte valeur ;
- 50 exemples obligatoires tracables.
