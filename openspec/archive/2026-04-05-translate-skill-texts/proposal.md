# Proposition

## Titre

Traduire en anglais les textes des skills Git portables

## Motivation

Le depot cible des skills Git portables pour Codex, mais une partie des contenus textuels des skills, README de skills, references, gabarits et scenarios pouvait etre en francais ou dans un etat linguistique mixte.

## Portee

- traduction en anglais des textes des skills Git si necessaire ;
- verification de coherence linguistique ;
- consolidation de la politique de langue dans la specification active.

## Resolution

L audit a confirme que les skills Git ciblees etaient deja en anglais. Le changement a donc ete resolu par verification et formalisation OpenSpec, sans traduction supplementaire des skills.
